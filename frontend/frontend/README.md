# 🧾 Automated Vendor Settlement System

A backend system designed to automate vendor payments, manage transactions, and generate settlement reports efficiently.

---

## 🚀 Project Overview
This system helps in managing vendors and automating their settlement process by tracking orders, calculating payouts, and maintaining financial records in a structured database.

---

## ⭐ Key Features
- Vendor management system
- Automated settlement calculation
- Transaction tracking and history
- Secure authentication system
- MySQL database integration
- REST API architecture

---

## 🛠️ Tech Stack
- Backend: Python (FastAPI / Flask)
- Database: MySQL
- ORM: SQLAlchemy
- Data Processing: Pandas
- Authentication: JWT
